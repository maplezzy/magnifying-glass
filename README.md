# magnifying-glass
图片放大镜组件

# 安装my-magnifying-glass

```shell
npm i my-magnifying-glass
```

# 引入

```js
import MagnifyingGlass from 'my-magnifying-glass';
```



# 使用

```vue
<magnifying-glass :imageSrc="xxxx" />
```
